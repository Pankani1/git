    Git Commands

    This is the third paragraph

    • git init: initializes a new Git repository.
    • git status: displays the status of the repository.
    • git log: displays a log of all commits made to the repository.
    • git add 'filename': stages a file for the next commit.
    • git add: stages all changes in the repository.
    • git commit -m "message": commits changes with a specified message.
    • git remote add origin_repo-url: adds a remote repository.
    • git push -u origin master: pushes changes to the remote repository.
    • git branch branchname: creates a new branch.
    • git branch: lists all branches.
    • git merge: merges changes from another branch.
    • git pull: fetches and merges changes from the remote repository.
    • git rm --cached filename : removes files.
    • git checkout :

    Command we will be using often

    • git status
    • git add
    • git commit -m "message"
    • git push -u origin master
